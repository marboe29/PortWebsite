Instructions on how to run and view your visualisation

File installed: Navigate to the Dashboard tab located on the excel file at the bottom, select both the dropdowns and option buttons to change between views that will change charts & fields automatically based on selected.

Section 1 – Charts: Top 5 industry’s & Emission types by industry

- Top 5 industries selectable fields are 7 types of emissions: Carbon dioxide, Fluorinated gases, Hydrofluorocarbons, Methane, Nitrous oxide, Perfluorocarbons, Sulphur hexafluoride. Number fields located under chart showing total & change amount for all listed industries on current chart. Industries are ranked by highest amount for the selected emission type. Including showing the past data these industries have seen from 07 to 19.

- Emission types by industry showing the top-ranking Industries from the previous chart but an in-depth look into its emission types by industry as an overall % across them that’s contributing to the amount. Selectable buttons are groups 1-3 which include the top 12 industries in total.

Section 2 – Charts: Total emissions by industry & by year

- Total by industry revealing the emission types and amount over the period 07-19 as sum, listing 20+ industries it shows the true scale of the emission types showing the amount rather than overall % within the industry.

- Total by year includes all industries by emission type for each given year out of the time frame, showing the range, trend and increments amounts for each type. Number fields relating to this chart are displayed bottom left that will select all 7 emission types, on change effect will come from the dropdown chart top 5, calculating total for each emission type, average for that type and the overall % out of the 7 emission types as rank.

How to address the data visualization

1. Questions that warrant the use of visualization methods

how much emissions come from each industry. what is the overall amount and averages for each emission. how many emission types are within each industry. what industries have increased. how the industry emissions have scaled over time. what emission is ranked highest among the industries. what emissions have increased.

2. Description of the source and type of dataset used

Stats NZ official data agency is a government department. From the Ministry for the Environment and show production-based gross collects information from people and organisations through censuses and surveys.

Greenhouse gas emissions (industry and household): Year ended 2019. (Government, 2021) CSV format

3. Visualization techniques evaluated, selected and used

Clustered bar chart: A horizontal view both for width of chart and text lineament, a dropdown to reduce to a single chart but its fields to change for multiple charts saving space, stacked bars showing any increments over time for each type.
Pie chart: Grouped charts displaying more info but within a good chart size as a whole, a selector to change between multiple groups of charts saving space, A circular view showing a proportional to the fraction of the whole in each type.
Area chart: A single chart scaling across the total for each category by type, volume-based view including each type by colour code for a single category while scaling to the most on one end.
Stacked line chart: A single chart scaling across the total for each year by type, to include all types in a stacked view listing topmost ranked type for comparison as well as any increments over time.

Number fields as extra values for input based from the charts that’s not visible unless calculated from the chart, to display different values for each type once selected from the dropdown.

4. Values the visualization provides for a given context

Air pollutants responsible for environmental effects such as climate, air pollutions, extreme weather, crop fields and increased wildfires. Other effects such as respiratory diseases, food supply disruptions and reduced atmospheric visibility.
